Cuplikan tampilan untuk landing page (section Cuplikan fitur).

File PNG yang dipakai halaman (sesuai index.html):
  - mbg-portal-desktop.png   → portal Staff / Relawan / Supplier (layar lebar)
  - mbg-app-login.png        → portal staff — layar login aplikasi dapur
  - mbg-supplier-portal.png  → portal supplier & pengadaan
  - mbg-staff-portal.png     → portal relawan & absensi
  - mbg-portal-mobile.png    → portal pilih (tampilan ponsel)

Cara memperbarui:
1. Ambil screenshot baru (disarankan lebar 1200–1600px untuk desktop; PNG atau WebP).
2. Simpan di folder ini dengan nama file yang sama persis, ATAU ubah src dan data-full di landing/index.html bagian Cuplikan fitur.

File SVG lama (ringkasan-ops.svg, produksi.svg, dll.) masih ada sebagai cadangan ilustrasi; halaman utama memakai PNG di atas.
